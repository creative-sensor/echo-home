### Git
- Open Git window: ```:Git```
- Toggle file to/from stage: ```s```
- New commit: ```cc```
- Amend commit: ```ca```
- git-diff: ```=```
- git-log: ```:GV```
- Push: ```tty    gpush```
- Push forced: ```tty    gforce```


### Buffers
- List buffer : ```\bs```
- Go Previous: ```Bp```
- Go Next: ```Bn```

### File explorer
- Open file explorer: ```:FgXp .```
- Move window to new tab: ```Ctrl w T```


### Misc
- Handy shell: ```tty```
- Grep text: ```:Grin ${TEXT}```
- Print flat path: ```:Jpath``` , ```:Ypath```
