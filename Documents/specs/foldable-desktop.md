# DEFINITION

- `foldable-desktop` is a `Foldable` desktop


# STRUCTURE

- `Cable`: dual type-c with display-port mode and power delivery
- `Screen`: 16inch with type-c port for display-port and power input
- `Sound`: Bluetooth headset or earbuds
- `Keyboard`: Bluetooth
- `Mouse`: Bluetooth+WirelessUSB
- `Desktop`: `13cm x 13cm x 6cm` or `5inch x 5inch x 2inch`
- `Power Adaptor`: Type-C or DC
