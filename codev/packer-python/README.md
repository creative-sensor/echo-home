# HOW TO BUILD PYTHON PACKAGE

### EDIT VARS
```./VARS```

### PUT SOURCE CODE
```
src/
├── sub_pkg_a
│   ├── car.py
│   └── __init__.py
└── sub_pkg_b
    ├── aircraft.py
    ├── __init__.py
    └── sub_pkg_x
        ├── beercraft.py
        └── __init__.py
```

### CREATE WORKSPACE
```
.function/workspace
```

### BUILD PACKAGE
```
.function/build
```
### INSTALL PACKAGE LOCAL
```
.function/install-local
```

### TEST PACKAGE
```
.function/test
```
