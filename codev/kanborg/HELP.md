### KEY GUIDE
- ```--```  view card
- ```---``` edit card
- ```mmm``` move card to the right
- ```bbb``` move card to the left

### MARK GUIDE

- insert media
```
![NAME.svg](./.media/NAME.svg)

```
<img alt="" src=".res/carbon.png"/>
```

- insert media url
```
