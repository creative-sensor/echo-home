# Rust Artifact Repository

This directory mirrors the artifact management structure of the Python environment, tailored for Rust toolchains and packages managed via Cargo.

## Structure Overview
- `.properd/`: Contains metadata, inventory (`artefacts.yaml`), and checksums.
- `.function/`: Contains executable scripts for artifact management (get, geti, readi, seal, etc.).
- `artefacts/`: Stores downloaded and processed packages.
